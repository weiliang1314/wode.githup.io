防抖函数

描述：当持续触发事件时，n秒内没有再触发事件，事件处理函数才会执行一次，如果设定的时间到来之前，又一次触发了事件，就重新开始延时。
用途：搜索联想，用户在不断输入值时，用防抖来节约请求资源。

代码实现

let input = document.querySelector('input');

function debounce(func, delay) {
let timer = null;

 return function(...args) {
    if (timer) clearTimeout(timer);

    timer = setTimeout(() => {
        //有参数时
        func.apply(this, args); //func();
           }, delay)
        }
   }

节流函数

描述：连续触发事件但是在n秒中只执行一次函数。
用途：鼠标连续不断地触发某事件，只在单位时间内只触发一次；在页面的无限加载场景下，需要用户在滚动页面时，每隔一段时间发一次请求，而不是在用户停下滚动页面操作时才去请求数据。

代码实现

function throttle(fn, delay) {
let timer;
return function(...args) {
    if (timer) {
        return;
       }
    timer = setTimeout(() => {
        // fn.apply(this, arguments);或
        //三点运算符，代替arguments:传递给函数的隐式参数，调用传递给函数fn实参
        fn(...args);

        timer = null;
           }, delay)
         }

     }

打家劫舍问题（动态规划）

问题：给定一个数组，代表每个房屋存放的金额，当偷相邻两个房屋时就会报警，计算在不触发报警系统的情况下偷盗的最大金额
问题分析：数组中的值为1个或0个则问题不成立，假设数组中有3个以上数值，所取得数不能相邻，当前索取金额总数=正在取+之前取,所取得金额总数最大;m[i]=n[i]+m[i-2]或m[i]=mi-1

代码实现

  let rob = function(nums) {
    //边界
    if (nums.length === 0) {
        return 0
    }
    if (nums.length === 1) {
        return nums[0]
    }
    const memo = [];
    memo[0] = nums[0];
    memo[1] = Math.max(nums[0], nums[1]);
    for (let i = 2; i <= nums.length; i++) {
        memo[i] = Math.max(nums[i] + memo[i - 2], memo[i - 1]);

    }
    return memo[nums.length - 1];
}

螺旋矩阵问题

问题：输入一个二维数组，按照从左向右，再向下，再向左，再向上，向右的方式遍历输出
问题分析：因为输入二维数组，按照顺时针遍历输出，那么首先就要要考虑边界问题；left最开始0；right为m0.length-1(m为输入的二维数组)，top为0，bottom为m.length-1;从左向右遍历不能超过右边界，从上到下不能超过下边界

代码实现

   let spiralOrder = function(matrix) {
  if (matrix.length == 0) {
    return [];
  }
  let left = 0;
  let right = matrix[0].length - 1;
 let top = 0;
  let bottom = matrix.length - 1;
  let dir = "right";
  let result = [];
  while (left <= right && top <= bottom) {
    if (dir == "right") {
        for (let i = left; i <= right; i++) {
            result.push(matrix[top][i])
        }
        top++;
        dir = "down"
    } else if (dir == "down") {
        for (let i = top; i <= bottom; i++) {
            result.push(matrix[i][right])
        }
        right--;
        dir = 'left';
    } else if (dir == "left") {
        for (let i = right; i >= left; i--) {
            result.push(matrix[bottom][i])
        }
        bottom--;
        dir = "top"
    } else if (dir == "top") {
        for (let i = bottom; i >= top; i--) {
            result.push(matrix[i][left])
        }
        left++;
        dir = "right"
    }
}
return result;

}

加油站问题（贪心）

贪心算法：从某一个初始状态出发，每次通过选取局部最优解向目标前进，并最终期望取得整体最优解的一种算法。由这个定义可知，贪心选择标准就是选择“当前最好”的决策，贪心算法根据这个标准进行决策，将原问题变成一个相似但规模更小的子问题，而后每一步选出来的一定是原问题整体最优解的一部分。如果一个问题贪心后只剩下一个子问题且有最优子结构，那么该问题就可以使用贪心算法。当一个问题的整体最优解包含其子问题的最优解时，我们称次问题具有最优子结构性质。
问题：环形路上有n个加油站，其中第i个加油站有get[i]升油， 有一辆无限容量油箱的车，从i加油站到i+1需要消耗cos[i]升油从其中一个加油站出发，开始时油箱没油，环形一周，返回出发时加油站编号
问题分析：首先要考虑所有加油站的油量是否多于或等于整个路程所消耗的油量；在此问题中假设汽车开始位置s,当前油量c，汽车开始时油量
为0；判断从i加油站到i+1加油站汽车油量是否足够，行驶后油量：c=c+get[i]-cos[i]

代码实现

let canComolete = function(gas, cost) {
let total = 0; //加油站总油
let totalcost = 0; //消耗总油
for (let i = 0; i < gas.length; i++) {
    total += gas[i];
    totalcost += cost[i]
}
if (total < totalcost) {
    return -1
}
let current = 0;
let start = 0; //假设的初始加油站
for (let i = 0; i < gas.length; i++) {
    //当前油量
    current = current - cost[i] + gas[i];
    if (current < 0) {
        current = 0;
        start = i + 1;
    }
}
return start; //出发时加油站编号
   }

最大正方形

问题：输入一个0和1组成的二维矩阵，找到只包含1的最大正方形，返回面积
问题分析：当1在边界时，最大正方形面积为1，边长d=1;当所选1的左边，上边，左上边，都不是0，它的
边长才能加1，当前边长=上一次最短边长+1

代码实现

let maxzheng = function(martix) {
let maxSidelength = 0; //记录最大边长

 let dp = new Array(martix.length);
for (let i = 0; i < martix.length; i++) {
    dp[i] = new Array(martix[i].length).fill(0); //把它变成一个二维数组，初始值为0
    for (let j = 0; j < martix[i].length; j++) {
        if (martix[i][j] == '1') {
            if (i == 0 || j == 0) { dp[i][j] = 1; } else {
                dp[i][j] = Math.min(dp[i - 1][j], dp[i][j - 1], dp[i - 1][j - 1]) + 1 //左边，上边，左上
            }
            maxSidelength = Math.max(maxSidelength, dp[i][j])

        }
          }
            }
          return maxSidelength * maxSidelength;
          }

什么是动态规划算法

动态规划常常适用于有重叠子问题和最优子结构性质的问题，动态规划方法所耗时间往往远少于暴力解法。大致上，若要解一个给定问题，我们需要解其不同部分（即子问题），再合并子问题的解以得出原问题的解。通常许多子问题非常相似，为此动态规划法只解决每个子问题一次，从而减少计算量，一旦某个给定子问题的解已经算出，则将其存储，以便下次需要同一个子问题解之时直接查表。

三要素

  1. 最优子结构：子问题的最优解能够决定这个问题的最优解
  2. 边界：问题最小子集的解
  3. 状态转移函数：递推式

    解决爬楼梯问题

问题：一共有n阶楼梯，每次只能爬一阶或两阶，有多少种爬的方法
分析与解决：在这个问题中我首先考虑到当n=1时f(1)=1,当n=2时f(2)=2,但当n>2时发生了变化，f(3)=3,f(4)=5…
由此得出公式：f(n)=f(n-1)+f(n-2) n>2;f(1)与f(2)相当于边界，公式就是状态转移函数，f(n-1)与f(n-2)就是最优子结构

代码实现

function pa(n) {
   if(n<1){return 0;}
    if (n == 1 || n == 2) {
      return n;
      }
     let w = [];
        w[0] = 1;
       w[1] = 2;
        for (let i = 2; i < n; i++) {
       w[i] = w[i - 1] + w[i - 2];
      }
        return w[n - 1];
       }

ajax实现基本流程

  1. 创建ajax对象
  2. 告诉请求方式与地址
  3. 发送请求
  4. 获取服务器端给客户端的数据

    封装注意的问题

1、请求参数位置：将请求参数传递到ajax函数内部，在函数内部根据请求方式不同将请求参数的位置放在不同位置
get 放在请求地址后面，post放在send()里面
2、请求参数格式 ：application/x-www-form-urlencoded（默认）,application/json 传递对象数据类型对于函数的调用者更加友好
3、如果请求方式是post，则需要在发送请求之前对请求的参数类型进行设置xhr.setRequestHeader(‘Content-Type’, contentType) json格式，需要把对象转换为json字符串
4、使用Object.assign()对设定默认值进行覆盖

封装代码

function ajax(options) {
     //默认值设置
     let defaults = {
             type: 'get',
             url: '',
             data: {},
             header: {
                 'Content-Type': 'application/x-www-form-urlencoded'
             },
             success: function() {},
             error: function() {}

         }
         //如果使用ajax时没有自定义，就使用默认值，如果使用了options就会覆盖默认值defults
         //对象覆盖方法
     Object.assign(defaults, options)

     //创建ajax对象
     let xhr = new XMLHttpRequest();
     //拼接请求参数的变量
     let params = '';
     for (let attr in defaults.data) {
         //将参数转换成字符串
         params += attr + '=' + defaults.data[attr] + '&'
     }
     //截取字符串最后面&
     params = params.substr(0, params.length - 1)
         //请求方式判断
     if (defaults.type == 'get') {
         defaults.url = defaults.url + '?' + params;
     }
     //配置对象
     xhr.open(defaults.type, defaults.url);
     //发送请求
     if (options.type == 'post') {
         let contentType = defaults.header['Content-Type']
             //请求参数类型设置

         xhr.setRequestHeader('Content-Type', contentType);
         if (contentType == 'application/json') {
             xhr.send(JSON.stringify(defaults.data))
         } else {
             xhr.send(params)
         }
     } else {
         xhr.send();
     }
     //当xhr对象接收完响应数据后触发，服务器端传过来的数据
     xhr.onload = function() {
         //获取响应头的数据
         let content = xhr.getResponseHeader('Content-Type')
             //服务器端返回的数据
         let responseText = xhr.responseText;
         //判断字符串是否包含某个字符
         if (content.includes('application/json')) {
             responseText = JSON.parse(responseText)
         }
         //请求成功http状态码
         if (xhr.status == 200) {
             //请求成功
             options.sucess(responseText, xhr);
         } else {
             //请求失败
             options.error(responseText, xhr)
         }
     }
   }

    对创建的集合进行插入数据

    1、普通方法
    //创建集合,名字首字母大写，应用相应属性规则
 const Course = mongoose.model('Course', course)
  //插入数据
 const cour = new Course({
  name: '小辣椒',
  author: '你写',
  ispub: true

   })
  //保存到数据库
  cour.save()
    2、create()方法，返回promise对象
    Course.create({
 name: 'javascript',
 author: 'hahaha',
 ispub: true
   }, (err, doc) => {
  //错误信息
  console.log(err)
     //插入的信息
  console.log(doc)
  })

    对集合中的数据进行删除

    1、根据条件删除单条数据，如果有多条数据符合条件则按顺序删除第一条，返回删除的数据
    Course.findOneAndDelete({ 条件}).then(res => { console.log(res) })
    2、根据条件删除多条条数据，如果不写条件就全部删除，返回删除的数据
    Course.deleteMany({}).then(res => { console.log(res)})

    对集合中数据进行修改

    1、修改单条数据
    Course.updateOne({ 条件}, { 要修改的数据}).then(res => { console.log(res) })
    2、修改多条数据，若没有条件就是修改所有
    Course.updateMany({}, {要修改的数据}).then(res => { console.log(res) })

    对集合中的数据进行查询

    1、查询所有数据
    Course.find().then(res => console.log(res))
    2、按条件查询多条数据
    Course.find({条件}).then(res => console.log(res))
    3、按条件查询单条数据
    Course.findOne({ 条件}).then(res => console.log(res))
    4、按字段查询数据
    Course.find().select('字段1 字段2...').then(res => console.log(res))
    5、skip()跳过多少条数据，limit()限制查询数据的数量
     //跳过前两条数据，查询两条数据
Course.find().skip(2).limit(2).then(res => console.log(res))

    mongoDB介绍

    MongoDB 是由C++语言编写的，是一个基于分布式文件存储的开源数据库系统。在高负载的情况下，添加更多的节点，可以保证服务器性能，为WEB应用提供可扩展的高性能数据存储解决方案。MongoDB 将数据存储为一个文档，数据结构由键值(key=>value)对组成。MongoDB 文档类似于JSON 对象。字段值可以包含其他文档，数组及文档数组。
