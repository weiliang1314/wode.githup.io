Nuxt.js概念

Nuxt.js简单的说是Vue.js的通用框架，最常用的就是用来作服务器端渲染；是一个专注于ui渲染的应用框架，可以快速搭建项目

Nuxt.js使用及介绍

构建项目结构

npm create nuxt-app

官网

https://nuxtjs.org/docs/2.x/get-started/installation

生命周期

1. nuxtServerlnit()
2. middleware()
3. validate()
4. asyncData()
5. Render()

约定路由

 <template>
<nav>
  <el-button type='primary'>点击</el-button>
  <nuxt-link to="/index" active-class="app">首页</nuxt-link>
    <nuxt-link to="/login" active-class="app">登录</nuxt-link>
    <nuxt-link to="/user" active-class="app">用户</nuxt-link>
    <nuxt-link to="/goods" active-class="app">商品</nuxt-link></nav>
     <nuxt />
</template>
