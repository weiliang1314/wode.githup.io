什么是动态规划算法

动态规划常常适用于有重叠子问题和最优子结构性质的问题，动态规划方法所耗时间往往远少于暴力解法。大致上，若要解一个给定问题，我们需要解其不同部分（即子问题），再合并子问题的解以得出原问题的解。通常许多子问题非常相似，为此动态规划法只解决每个子问题一次，从而减少计算量，一旦某个给定子问题的解已经算出，则将其存储，以便下次需要同一个子问题解之时直接查表。

三要素

  1. 最优子结构：子问题的最优解能够决定这个问题的最优解
  2. 边界：问题最小子集的解
  3. 状态转移函数：递推式

    解决爬楼梯问题

问题：一共有n阶楼梯，每次只能爬一阶或两阶，有多少种爬的方法
分析与解决：在这个问题中我首先考虑到当n=1时f(1)=1,当n=2时f(2)=2,但当n>2时发生了变化，f(3)=3,f(4)=5…
由此得出公式：f(n)=f(n-1)+f(n-2) n>2;f(1)与f(2)相当于边界，公式就是状态转移函数，f(n-1)与f(n-2)就是最优子结构

代码实现

function pa(n) {
   if(n<1){return 0;}
    if (n == 1 || n == 2) {
      return n;
      }
     let w = [];
        w[0] = 1;
       w[1] = 2;
        for (let i = 2; i < n; i++) {
       w[i] = w[i - 1] + w[i - 2];
      }
        return w[n - 1];
       }

解决最低票价问题

问题：假设旅游时,有三种票价：玩一天costs[0]元，连续玩七天costs[1]元，连续玩三十天costs[2]元，一年365天数组days相当于每天1~365，dyas[]中的天数是有旅游的，自定义函数返回days[]中每一天最低票价。
问题分析：

代码实现

function mincost(days, costs) {
  let dp = new Array(days[days.length - 1] + 1).fill(0); //使用固定值替换数组元素初始化全为0，dp[0]=0没有旅行
     let daysIndex = 0;
          for (let i = 1; i < dp.length; i++) {
             if (i === days[daysIndex]) {
           dp[i] = Math.min(
          dp[Math.max(0, i - 1)] + costs[0],
          //在i-7天买了一张七天票，现在还能用,在i-7这一天只买了还没用
          dp[Math.max(0, i - 7)] + costs[1],
          dp[Math.max(0, i - 30)] + costs[2],
              )
                 daysIndex++;
           } else {
            //没有费用
              dp[i] = dp[i - 1]
            }
             }
          return parseInt(dp[dp.length - 1]) //完成当天以及之前天，花费最少费用
             }
       console.log(mincost([1, 4, 6, 7, 8, 20], [2, 7, 15]))//11
