vue-cli使用

title: vue/cli使用
date: 2020-07-14 18:56:12
tags: 关于vue-cli2的一些基本使用

###基于node.js与webpack的vue/cli全局安装
npm install -g @vue/cli@版本号（可以不写选择默认安装新版本）
###安装完成后用来创建项目
vue init webpack 项目名称
###项目初始化过程中的一些配置选择
1、名称，项目描述，作者可以选择默认；
2、runtime-compiler与runtime-only的选择：建议选择后者，相比前者性能更好,代码更少
程序运行区别：
runtime-complier
template->ast->render->vdom->UI
runtime-only
render->vdom->UI
代码区别：
runtime-only
new Vue({
el: ‘#app’,
router,
render: h => h(App)
})
runtime-complier
new Vue({
el: ‘#app’,
components: { App },
template: ‘‘
})
3、选择安装路由：vue-router
4、选择Eslint代码规范
5、单元测试
6、端到端测试
7、项目管理工具：npm或yarn
