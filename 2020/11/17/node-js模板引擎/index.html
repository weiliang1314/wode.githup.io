art-template介绍

采用作用域预声明的技术来优化模板渲染速度，从而获得接近 JavaScript 极限的运行性能，并且同时支持 NodeJS 和浏览器；拥有接近 JavaScript 渲染极限的的性能，支持 Express、Koa、Webpack，支持模板继承与子模板

在项目中使用下载

下载 npm install art-template
使用
 const template = require('art-template');

const views = path.join(__dirname, 'views', '模板文件名')
 const html = template(views, {
    用到的数据
      })
