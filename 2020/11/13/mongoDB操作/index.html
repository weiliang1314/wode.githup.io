mongoDB操作

title: mongoDB操作
date: 2020-11-13 20:16:28
tags: 使用node.js对mongoDB实现增、删、改、插操作

对创建的集合进行插入数据

1、普通方法
//创建集合,名字首字母大写，应用相应属性规则
 const Course = mongoose.model('Course', course)
  //插入数据
 const cour = new Course({
  name: '小辣椒',
  author: '你写',
  ispub: true

   })
  //保存到数据库
  cour.save()
2、create()方法，返回promise对象
Course.create({
 name: 'javascript',
 author: 'hahaha',
 ispub: true
   }, (err, doc) => {
  //错误信息
  console.log(err)
     //插入的信息
  console.log(doc)
  })

对集合中的数据进行删除

1、根据条件删除单条数据，如果有多条数据符合条件则按顺序删除第一条，返回删除的数据
Course.findOneAndDelete({ 条件}).then(res => { console.log(res) })
2、根据条件删除多条条数据，如果不写条件就全部删除，返回删除的数据
Course.deleteMany({}).then(res => { console.log(res)})

对集合中数据进行修改

1、修改单条数据
Course.updateOne({ 条件}, { 要修改的数据}).then(res => { console.log(res) })
2、修改多条数据，若没有条件就是修改所有
Course.updateMany({}, {要修改的数据}).then(res => { console.log(res) })

对集合中的数据进行查询

1、查询所有数据
Course.find().then(res => console.log(res))
2、按条件查询多条数据
Course.find({条件}).then(res => console.log(res))
3、按条件查询单条数据
Course.findOne({ 条件}).then(res => console.log(res))
4、按字段查询数据
Course.find().select('字段1 字段2...').then(res => console.log(res))
5、skip()跳过多少条数据，limit()限制查询数据的数量
 //跳过前两条数据，查询两条数据
Course.find().skip(2).limit(2).then(res => console.log(res))

同一个数据库对两个数据集合进行关联

连接好数据库后，建立两个数据集合及其规则

const user = new mongoose.Schema({
 name: {
    type: String,
    required: true
}
   })
 const post = new mongoose.Schema({
    title: {
       type: String
     },
  //对两个表进行关联
      author: {
    type: mongoose.Schema.Types.ObjectId,
   ref: 'Users'
     }
    })
 const Users = mongoose.model('Users', user)
 const Posts = mongoose.model('Posts', post)

插入数据时把关联的字段上写被关联字段数据的id

Users.create({
 name: '朱自清'
  }, ).then(doc => console.log(doc)).catch(err => {
 console.log(err)
})

title: '荷塘月色 背影',
 author: '5fa698057b04462b6031e4f7'//name的id
  }, ).then(doc => console.log(doc)).catch(err => {
  console.log(err)
